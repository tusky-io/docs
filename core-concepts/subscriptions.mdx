Tusky offers three plans. All plans include the same core features — the difference is scale. Walrus storage costs (WAL) are always billed separately from your [balance](/core-concepts/balance) regardless of plan.

## Plans

|  | **Pay As You Go** | **Developer** | **Business** |
| --- | :-: | :-: | :-: |
| **Monthly price** | \$0 | \$50 | \$500 |
| **Storage included** | 5 GB | 50 GB | 2 TB |
| **Transactions / month** | 1000 | 10 000 | 100 000 |
| **Bandwidth / month** | 50 GB | 500 GB | 10 TB |
| **Aggregators** | 1 private aggregator | 1 private aggregator | 3 private aggregators<br />with custom domains |
| Backup |  |  |  |
