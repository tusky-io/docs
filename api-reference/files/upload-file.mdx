---
title: Upload File
api: 'POST https://api.tusky.io/v2/vaults/{vaultId}/files'
---

Upload a file to a vault. Files are stored on the Walrus decentralized storage network. For encrypted vaults, encryption is handled automatically before storage.

The request must use `multipart/form-data` encoding.

## Path Parameters

<ParamField path="vaultId" type="string" required>
  The unique identifier of the vault to upload the file to.
</ParamField>

## Request Body

<ParamField body="file" type="binary" required>
  The file to upload. Maximum file size is **2.5 GB**.
</ParamField>

<ParamField body="epochs" type="number" default="5">
  Number of Walrus epochs to store the file. Each epoch is approximately 24 hours. Higher values increase storage duration and cost.
</ParamField>

<ParamField body="autoExtend" type="boolean" default="false">
  Whether to automatically extend storage before the file expires. When enabled, Tusky will renew the file's storage at the end of each epoch.
</ParamField>

## Response

<ResponseField name="uploadId" type="string">
  Unique identifier for the upload operation.
</ResponseField>

<ResponseField name="id" type="string">
  Unique identifier for the file.
</ResponseField>

<ResponseField name="name" type="string">
  Original filename.
</ResponseField>

<ResponseField name="mimeType" type="string">
  MIME type of the uploaded file.
</ResponseField>

<ResponseField name="size" type="number">
  File size in bytes.
</ResponseField>

<ResponseField name="blobId" type="string">
  Walrus blob identifier for the stored file.
</ResponseField>

<ResponseField name="quiltPatchId" type="string">
  Quilt patch identifier for the file metadata on Sui.
</ResponseField>

<ResponseField name="vaultId" type="string">
  The vault this file belongs to.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the file was uploaded.
</ResponseField>

<ResponseField name="expiresAt" type="string">
  ISO 8601 timestamp of when the file's storage expires.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.tusky.io/v2/vaults/vault_abc123def456/files \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@./document.pdf" \
  -F "epochs=10" \
  -F "autoExtend=true"
```

</CodeGroup>

### Response

```json
{
  "uploadId": "upload_xyz789",
  "id": "file_mno345pqr678",
  "name": "document.pdf",
  "mimeType": "application/pdf",
  "size": 1048576,
  "blobId": "walrus_blob_abc123",
  "quiltPatchId": "qp_def456",
  "vaultId": "vault_abc123def456",
  "createdAt": "2025-07-01T12:30:00.000Z",
  "expiresAt": "2025-07-11T12:30:00.000Z"
}
```

<Note>
  The maximum file size is **2.5 GB**. For larger files, consider splitting them before upload.
</Note>

<Note>
  Files uploaded to encrypted vaults are encrypted client-side before being stored on Walrus. The encryption key is derived from the vault's configuration.
</Note>
