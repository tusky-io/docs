The Tusky API is a RESTful interface for managing decentralized storage on the [Sui Walrus](https://walrus.xyz) network. Use it to create vaults, upload and download files, manage access control, configure publishers, and control your aggregators.

## Base URL

```
https://api.tusky.io/v2
```

All API requests must be made over HTTPS. Requests made over plain HTTP will be rejected.

## Content Type

All endpoints accept and return JSON (`application/json`) unless otherwise noted. File uploads use `multipart/form-data`.

## Request format

Include your API key in the `Authorization` header of every request:

```bash
curl https://api.tusky.io/v2/vaults \
  -H "Authorization: Bearer YOUR_API_KEY"
```

See [Authentication](/api-reference/authentication) for details on generating and managing API keys.
