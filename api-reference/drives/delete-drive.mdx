---
title: Delete Drive
api: 'DELETE https://api.tusky.io/v2/drives/{driveId}'
---

Permanently delete a drive and all of its files.

## Path Parameters

<ParamField path="driveId" type="string" required>
  The unique identifier of the drive to delete.
</ParamField>

## Response

Returns a `204 No Content` response on success.

## Example

<CodeGroup>

```bash cURL
curl -X DELETE https://api.tusky.io/v2/drives/drive_abc123def456 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>

### Response

```
204 No Content
```

<Warning>
  This action is **irreversible**. Deleting a drive permanently removes the drive and all files within it. File data stored on the Walrus network may persist until the epoch expiry, but will no longer be accessible through Tusky.
</Warning>
