---
title: Get Drive
api: 'GET https://api.tusky.io/v2/drives/{driveId}'
---

Retrieve details of a specific drive by its ID.

## Path Parameters

<ParamField path="driveId" type="string" required>
  The unique identifier of the drive.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the drive.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the drive.
</ResponseField>

<ResponseField name="encrypted" type="boolean">
  Whether the drive is encrypted.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the drive was created.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of when the drive was last updated.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/drives/drive_abc123def456 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>

### Response

```json
{
  "id": "drive_abc123def456",
  "name": "My First Drive",
  "encrypted": false,
  "createdAt": "2025-07-01T12:00:00.000Z",
  "updatedAt": "2025-07-01T12:00:00.000Z"
}
```
