---
title: Get Gateway
api: 'GET https://api.tusky.io/v2/gateway'
---

Retrieve the authenticated user's private gateway configuration. Private gateways provide a dedicated URL for accessing vault files.

## Response

<ResponseField name="url" type="string">
  The full URL of the private gateway.
</ResponseField>

<ResponseField name="subdomain" type="string">
  The subdomain assigned to the gateway (e.g., `myapp` for `myapp.gateway.tusky.io`).
</ResponseField>

<ResponseField name="customDomain" type="string">
  Custom domain configured for the gateway, if any. `null` if not configured.
</ResponseField>

<ResponseField name="analyticsEnabled" type="boolean">
  Whether gateway analytics are enabled.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/gateway \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>

### Response

```json
{
  "url": "https://myapp.gateway.tusky.io",
  "subdomain": "myapp",
  "customDomain": "files.example.com",
  "analyticsEnabled": true
}
```
