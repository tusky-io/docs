---
title: Get Secret
api: 'GET https://api.tusky.io/v2/secrets/{path}'
---

Read a secret at the specified path. The encrypted data is fetched from the Seal network and decrypted client-side using your wallet's keys.

## Path Parameters

<ParamField path="path" type="string" required>
  The path of the secret to read (e.g., `production/database`).
</ParamField>

## Query Parameters

<ParamField query="version" type="number">
  Specific version to retrieve. Defaults to the latest version.
</ParamField>

## Response

<ResponseField name="path" type="string">
  The secret path.
</ResponseField>

<ResponseField name="data" type="object">
  Key-value pairs containing the secret data.
</ResponseField>

<ResponseField name="version" type="number">
  The version number of this secret.
</ResponseField>

<ResponseField name="environmentId" type="string">
  Associated environment ID, or `null` if account-level.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when this version was created.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/secrets/production/database \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>
