---
title: Delete Secret
api: 'DELETE https://api.tusky.io/v2/secrets/{path}'
---

Delete a secret at the specified path. All versions are removed.

**Managed secrets** (paths under `tusky/`) **cannot be deleted** — the API returns `403 Forbidden`.

## Path Parameters

<ParamField path="path" type="string" required>
  The path of the secret to delete.
</ParamField>

## Response

Returns `204 No Content` on success.

## Example

<CodeGroup>

```bash cURL
curl -X DELETE https://api.tusky.io/v2/secrets/production/database \
  -H "Api-Key: YOUR_API_KEY"
```

</CodeGroup>

<Warning>
  Deletion is permanent and removes all versions. The Seal on-chain encryption keys are revoked, making any cached encrypted copies unreadable.
</Warning>

<Note>
  Managed secrets like `tusky/volumes/{envId}/encryption` are non-deletable. Deleting the parent volume is the only way to remove them.
</Note>
