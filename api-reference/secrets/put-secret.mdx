---
title: Put Secret
api: 'PUT https://api.tusky.io/v2/secrets/{path}'
---

Create or update a secret at the specified path. The secret data is encrypted using [Sui Seal](https://docs.mystenlabs.com/seal) and stored on-chain — Tusky never has access to the plaintext values.

## Path Parameters

<ParamField path="path" type="string" required>
  The path to store the secret at. Use `/` separators for hierarchy (e.g., `production/database/password`).
</ParamField>

## Request Body

<ParamField body="data" type="object" required>
  Key-value pairs to store as the secret. Values are strings.
</ParamField>

<ParamField body="driveId" type="string">
  Drive to associate the secret with. Secrets inherit the drive's access policies. If omitted, the secret is stored at the account level.
</ParamField>

## Response

<ResponseField name="path" type="string">
  The full path of the stored secret.
</ResponseField>

<ResponseField name="version" type="number">
  The version number of this secret. Increments on each update.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when this version was created.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X PUT https://api.tusky.io/v2/secrets/production/database \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "data": {
      "host": "db.example.com",
      "port": "5432",
      "username": "app_user",
      "password": "s3cur3_p4ssw0rd"
    }
  }'
```

</CodeGroup>

### Response

```json
{
  "path": "production/database",
  "version": 1,
  "createdAt": "2026-03-06T14:00:00Z"
}
```

<Note>
  Secrets are encrypted client-side using keys managed by [Sui Seal](https://docs.mystenlabs.com/seal). Access is controlled by on-chain policies — only wallets authorized by the Seal policy can decrypt the secret data.
</Note>

<Tip>
  Use path hierarchy to organize secrets logically: `{environment}/{service}/{key}`. This makes it easy to list and manage secrets per environment or service.
</Tip>
