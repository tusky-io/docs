---
title: List Folders
api: 'GET https://api.tusky.io/v2/drives/{driveId}/folders'
---

List folders in a drive.

## Path Parameters

<ParamField path="driveId" type="string" required>
  The drive ID.
</ParamField>

## Query Parameters

<ParamField query="parentId" type="string">
  Filter by parent folder. Use `null` or omit to list root-level folders.
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor.
</ParamField>

<ParamField query="limit" type="number" default="20">
  Items per page. Max `100`.
</ParamField>

## Example

<CodeGroup>

```bash cURL
curl "https://api.tusky.io/v2/drives/drive_abc123/folders" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>

### Response

```json
{
  "data": [
    {
      "id": "folder_xyz789",
      "name": "Photos",
      "driveId": "drive_abc123",
      "parentId": null,
      "createdAt": "2026-03-06T12:00:00Z"
    }
  ],
  "nextCursor": null,
  "hasMore": false
}
```
