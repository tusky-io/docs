---
title: Create Folder
api: 'POST https://api.tusky.io/v2/volumes/{volumeId}/folders'
---

Create a new folder inside a volume to organize files.

## Path Parameters

<ParamField path="volumeId" type="string" required>
  The volume to create the folder in.
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  Folder name.
</ParamField>

<ParamField body="parentId" type="string">
  Parent folder ID for nesting. Omit to create at the volume root.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique folder identifier.
</ResponseField>

<ResponseField name="name" type="string">
  Folder name.
</ResponseField>

<ResponseField name="volumeId" type="string">
  The volume this folder belongs to.
</ResponseField>

<ResponseField name="parentId" type="string">
  Parent folder ID, or `null` if at volume root.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 creation timestamp.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.tusky.io/v2/volumes/vol_abc123/folders \
  -H "Api-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "Photos", "parentId": null}'
```

</CodeGroup>

### Response

```json
{
  "id": "folder_xyz789",
  "name": "Photos",
  "volumeId": "vol_abc123",
  "parentId": null,
  "createdAt": "2026-03-06T12:00:00Z"
}
```
