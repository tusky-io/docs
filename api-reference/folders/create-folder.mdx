---
title: Create Folder
api: 'POST https://api.tusky.io/v2/drives/{driveId}/folders'
---

Create a new folder inside a drive to organize files.

## Path Parameters

<ParamField path="driveId" type="string" required>
  The drive to create the folder in.
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  Folder name.
</ParamField>

<ParamField body="parentId" type="string">
  Parent folder ID for nesting. Omit to create at the drive root.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique folder identifier.
</ResponseField>

<ResponseField name="name" type="string">
  Folder name.
</ResponseField>

<ResponseField name="driveId" type="string">
  The drive this folder belongs to.
</ResponseField>

<ResponseField name="parentId" type="string">
  Parent folder ID, or `null` if at drive root.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 creation timestamp.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.tusky.io/v2/drives/drive_abc123/folders \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "Photos", "parentId": null}'
```

</CodeGroup>

### Response

```json
{
  "id": "folder_xyz789",
  "name": "Photos",
  "driveId": "drive_abc123",
  "parentId": null,
  "createdAt": "2026-03-06T12:00:00Z"
}
```
