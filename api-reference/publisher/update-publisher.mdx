---
title: Update Publisher
api: 'PATCH https://api.tusky.io/v2/publisher'
---

Update the authenticated user's publisher configuration. Only the fields you include in the request body are updated — omitted fields remain unchanged.

## Request Body

<ParamField body="defaultEpochs" type="number">
  The default number of Walrus storage epochs for new uploads. Must be between 1 and 100. Each epoch is approximately 2 weeks.
</ParamField>

<ParamField body="publishingMode" type="string">
  How files are published to Walrus. Accepted values: `"quilts"` or `"blobs"`.
</ParamField>

<ParamField body="deletableBlobs" type="boolean">
  Whether newly published blobs should be marked as deletable on Walrus. When `true`, blobs can be deleted before their epochs expire.
</ParamField>

<ParamField body="autoExtend" type="boolean">
  Whether Tusky should automatically renew storage before epochs expire.
</ParamField>

<Warning>
  Changing `publishingMode` only affects **future uploads**. Files already published retain their original mode — quilts remain quilts and standalone blobs remain standalone blobs. This change cannot be applied retroactively.
</Warning>

## Response

<ResponseField name="defaultEpochs" type="number">
  The updated default storage epoch count.
</ResponseField>

<ResponseField name="publishingMode" type="string">
  The updated publishing mode (`"quilts"` or `"blobs"`).
</ResponseField>

<ResponseField name="deletableBlobs" type="boolean">
  Whether newly published blobs are marked as deletable.
</ResponseField>

<ResponseField name="autoExtend" type="boolean">
  Whether auto-extend is enabled.
</ResponseField>

<ResponseField name="managedWalletAddress" type="string">
  The Sui address of the managed wallet.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X PATCH https://api.tusky.io/v2/publisher \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "defaultEpochs": 5,
    "publishingMode": "blobs",
    "deletableBlobs": true,
    "autoExtend": true
  }'
```

```typescript SDK
import Tusky from "@tusky-io/ts-sdk";

const tusky = new Tusky({ apiKey: process.env.TUSKY_API_KEY });

const updated = await tusky.publisher.update({
  defaultEpochs: 5,
  publishingMode: "blobs",
  deletableBlobs: true,
  autoExtend: true,
});

console.log(updated);
```

</CodeGroup>

### Response

```json
{
  "defaultEpochs": 5,
  "publishingMode": "blobs",
  "deletableBlobs": true,
  "autoExtend": true,
  "managedWalletAddress": "0x1a2b3c4d5e6f7890abcdef1234567890abcdef12"
}
```

<Note>
  To partially update the configuration, include only the fields you want to change. For example, to update just the epoch count:

  ```bash
  curl -X PATCH https://api.tusky.io/v2/publisher \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"defaultEpochs": 10}'
  ```
</Note>
