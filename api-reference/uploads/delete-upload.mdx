---
title: Delete Upload
api: 'DELETE https://api.tusky.io/v2/uploads/{uploadId}'
---

Cancel and delete an in-progress upload. Frees associated server resources. After deletion, all future requests to this upload URL return `404 Not Found` or `410 Gone`.

## Path Parameters

<ParamField path="uploadId" type="string" required>
  The upload ID to cancel.
</ParamField>

## Headers

<ParamField header="Tus-Resumable" type="string" required>
  Must be `1.0.0`.
</ParamField>

## Response

Returns `204 No Content` on success.

## Example

<CodeGroup>

```bash cURL
curl -X DELETE https://api.tusky.io/v2/uploads/upload_xyz789 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Tus-Resumable: 1.0.0"
```

</CodeGroup>

<Note>
  Incomplete uploads are automatically cleaned up after **24 hours**. Use this endpoint to free resources immediately if you know the upload won't be completed.
</Note>
