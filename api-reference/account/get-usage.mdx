---
title: Get Usage
api: 'GET https://api.tusky.io/v2/account/usage'
---

Retrieve usage statistics for the authenticated account.

## Query Parameters

<ParamField query="from" type="string">
  Start of the period (ISO 8601). Defaults to current billing period start.
</ParamField>

<ParamField query="to" type="string">
  End of the period (ISO 8601). Defaults to now.
</ParamField>

## Response

<ResponseField name="transactions" type="object">
  <Expandable title="Transaction metrics">
    <ResponseField name="total" type="number">Total transactions in the period.</ResponseField>
    <ResponseField name="thisMonth" type="number">Transactions in the current calendar month.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="bandwidth" type="object">
  <Expandable title="Bandwidth metrics">
    <ResponseField name="totalBytes" type="number">Total bandwidth in bytes.</ResponseField>
    <ResponseField name="thisMonthBytes" type="number">Bandwidth in the current calendar month.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="walletBalance" type="object">
  <Expandable title="Wallet balance">
    <ResponseField name="sui" type="string">SUI token balance.</ResponseField>
    <ResponseField name="wal" type="string">WAL token balance.</ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/account/usage \
  -H "Api-Key: YOUR_API_KEY"
```

</CodeGroup>
