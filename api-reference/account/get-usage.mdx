---
title: Get Usage
api: 'GET https://api.tusky.io/v2/account/usage'
---

Retrieve usage statistics for the authenticated account.

## Query Parameters

<ParamField query="from" type="string">
  Start of the period (ISO 8601). Defaults to 30 days ago.
</ParamField>

<ParamField query="to" type="string">
  End of the period (ISO 8601). Defaults to now.
</ParamField>

## Response

<ResponseField name="transactions" type="number">
  Total transactions (uploads, extends, deletes) in the period.
</ResponseField>

<ResponseField name="ingressBytes" type="number">
  Data uploaded in the period (bytes).
</ResponseField>

<ResponseField name="egressBytes" type="number">
  Data downloaded / served via aggregators in the period (bytes).
</ResponseField>

<ResponseField name="usedBytes" type="number">
  Total bytes currently stored on Walrus.
</ResponseField>

<ResponseField name="activeFiles" type="number">
  Number of active files.
</ResponseField>

<ResponseField name="expiredFiles" type="number">
  Number of files whose storage epochs have already expired.
</ResponseField>

<ResponseField name="environments" type="number">
  Number of environments on the account.
</ResponseField>

<ResponseField name="publishers" type="number">
  Number of publishers on the account.
</ResponseField>

<ResponseField name="aggregators" type="number">
  Number of aggregators on the account.
</ResponseField>

<ResponseField name="suiSpent" type="string">
  SUI spent in the period (gas fees).
</ResponseField>

<ResponseField name="walSpent" type="string">
  WAL spent in the period (Walrus storage).
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl "https://api.tusky.io/v2/account/usage?from=2026-03-01T00:00:00Z&to=2026-03-07T23:59:59Z" \
  -H "Api-Key: YOUR_API_KEY"
```

</CodeGroup>
