---
title: Get Usage
api: 'GET https://api.tusky.io/v2/account/usage'
---

Retrieve usage statistics for the authenticated account.

## Query Parameters

<ParamField query="from" type="string">
  Start of the period (ISO 8601). Defaults to 30 days ago.
</ParamField>

<ParamField query="to" type="string">
  End of the period (ISO 8601). Defaults to now.
</ParamField>

## Response

<ResponseField name="transactions" type="number">
  Total transactions (uploads, extends, deletes) in the period.
</ResponseField>

<ResponseField name="bandwidth" type="object">
  <Expandable title="Bandwidth metrics">
    <ResponseField name="ingressBytes" type="number">Data uploaded (bytes).</ResponseField>
    <ResponseField name="egressBytes" type="number">Data downloaded / served via aggregators (bytes).</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="storage" type="object">
  <Expandable title="Storage metrics">
    <ResponseField name="usedBytes" type="number">Total bytes currently stored on Walrus.</ResponseField>
    <ResponseField name="activeFiles" type="number">Number of active files.</ResponseField>
    <ResponseField name="environments" type="number">Number of environments.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="costs" type="object">
  <Expandable title="Cost breakdown">
    <ResponseField name="suiSpent" type="string">SUI spent in the period (gas + Tusky fees).</ResponseField>
    <ResponseField name="walSpent" type="string">WAL spent in the period (Walrus storage).</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="expiringFiles" type="number">
  Number of files with storage expiring within the next 2 epochs (~4 weeks).
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl "https://api.tusky.io/v2/account/usage?from=2026-03-01T00:00:00Z&to=2026-03-07T23:59:59Z" \
  -H "Api-Key: YOUR_API_KEY"
```

</CodeGroup>
