---
title: Get Usage
api: 'GET https://api.tusky.io/v2/account/usage'
---

Retrieve usage statistics for the authenticated account, including storage consumption, transaction counts, and costs.

## Response

<ResponseField name="storageUsed" type="number">
  Total storage used in bytes across all environments.
</ResponseField>

<ResponseField name="storageLimit" type="number">
  Maximum storage allowed under the current plan, in bytes.
</ResponseField>

<ResponseField name="filesCount" type="number">
  Total number of files stored across all environments.
</ResponseField>

<ResponseField name="environmentsCount" type="number">
  Total number of environments.
</ResponseField>

<ResponseField name="transactions" type="object">
  Transaction statistics for the current billing period.

  <Expandable title="Transaction properties">
    <ResponseField name="uploads" type="number">
      Number of file uploads.
    </ResponseField>

    <ResponseField name="downloads" type="number">
      Number of file downloads.
    </ResponseField>

    <ResponseField name="totalCost" type="string">
      Total cost in SUI tokens for the current billing period.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="billingPeriod" type="object">
  Current billing period information.

  <Expandable title="Billing period properties">
    <ResponseField name="start" type="string">
      ISO 8601 timestamp of the billing period start.
    </ResponseField>

    <ResponseField name="end" type="string">
      ISO 8601 timestamp of the billing period end.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/account/usage \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>
