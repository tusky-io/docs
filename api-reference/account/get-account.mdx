---
title: Get Account
api: 'GET https://api.tusky.io/v2/account'
---

Retrieve account details for the authenticated user.

## Response

<ResponseField name="id" type="string">Account ID.</ResponseField>
<ResponseField name="address" type="string">Connected Sui wallet address.</ResponseField>
<ResponseField name="managedWalletAddress" type="string">Tusky-managed wallet address for publishing.</ResponseField>
<ResponseField name="email" type="string">Email address, or `null` if not set.</ResponseField>
<ResponseField name="plan" type="string">Current plan: `payg`, `starter`, `growth`, `scale`, `enterprise`.</ResponseField>
<ResponseField name="createdAt" type="string">ISO 8601 creation timestamp.</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/account \
  -H "Api-Key: YOUR_API_KEY"
```

</CodeGroup>
