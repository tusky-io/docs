---
title: Get Account
api: 'GET https://api.tusky.io/v2/account'
---

Retrieve account details for the authenticated user, including wallet information and balance.

## Response

<ResponseField name="id" type="string">
  Unique identifier for the account.
</ResponseField>

<ResponseField name="email" type="string">
  The email address associated with the account.
</ResponseField>

<ResponseField name="walletAddress" type="string">
  The managed Sui wallet address for the account.
</ResponseField>

<ResponseField name="walletBalance" type="string">
  Current wallet balance in SUI tokens.
</ResponseField>

<ResponseField name="plan" type="string">
  The current billing plan (`free`, `pro`, `enterprise`).
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the account was created.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/account \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>
