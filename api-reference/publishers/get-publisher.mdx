---
title: Get Publisher
api: 'GET https://api.tusky.io/v2/publishers/{publisherId}'
---

Retrieve a publisher by ID.

## Path Parameters

<ParamField path="publisherId" type="string" required>
  The unique identifier of the publisher.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique publisher identifier.
</ResponseField>

<ResponseField name="active" type="boolean">
  Whether the publisher is active.
</ResponseField>

<ResponseField name="epochs" type="number">
  Default storage epoch count.
</ResponseField>

<ResponseField name="quilt" type="boolean">
  Whether quilt batching is enabled.
</ResponseField>

<ResponseField name="permanent" type="boolean">
  Whether blobs are non-deletable.
</ResponseField>

<ResponseField name="share" type="boolean">
  Whether blobs are shared.
</ResponseField>

<ResponseField name="driveId" type="string">
  The drive this publisher is attached to, or `null` if it is the account default.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 creation timestamp.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 last-updated timestamp.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/publishers/pub_7f3a2b1c \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>
