---
title: List Members
api: 'GET https://api.tusky.io/v2/volumes/{volumeId}/members'
---

List all members of a volume, including their roles.

## Path Parameters

<ParamField path="volumeId" type="string" required>
  The unique identifier of the volume.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of member objects.

  <Expandable title="Member object properties">
    <ResponseField name="id" type="string">
      Unique identifier for the membership.
    </ResponseField>

    <ResponseField name="address" type="string">
      The Sui wallet address of the member.
    </ResponseField>

    <ResponseField name="role" type="string">
      The assigned role (`viewer`, `manager`, or `owner`).
    </ResponseField>

    <ResponseField name="volumeId" type="string">
      The volume this membership belongs to.
    </ResponseField>

    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp of when the member was added.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/volumes/vol_abc123def456/members \
  -H "Api-Key: YOUR_API_KEY"
```

</CodeGroup>
