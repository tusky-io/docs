---
title: Remove Member
api: 'DELETE https://api.tusky.io/v2/drives/{driveId}/members/{memberId}'
---

Remove a member from a drive, revoking their access.

## Path Parameters

<ParamField path="driveId" type="string" required>
  The unique identifier of the drive.
</ParamField>

<ParamField path="memberId" type="string" required>
  The unique identifier of the member to remove.
</ParamField>

## Response

Returns a `204 No Content` response on success.

## Example

<CodeGroup>

```bash cURL
curl -X DELETE https://api.tusky.io/v2/drives/drive_abc123def456/members/member_stu901vwx234 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>

### Response

```
204 No Content
```

<Note>
  Only drive owners and admins can remove members. The drive owner cannot be removed. For encrypted drives, removing a member revokes their decryption access.
</Note>
