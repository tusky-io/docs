---
title: Add Member
api: 'POST https://api.tusky.io/v2/drives/{driveId}/members'
---

Add a new member to a drive by their Sui wallet address. Members can be assigned different roles that control their level of access.

## Path Parameters

<ParamField path="driveId" type="string" required>
  The unique identifier of the drive.
</ParamField>

## Request Body

<ParamField body="address" type="string" required>
  The Sui wallet address of the user to add as a member.
</ParamField>

<ParamField body="role" type="string" required>
  The role to assign to the member. One of:
  - `viewer` — read-only access to drive files
  - `contributor` — can upload, download, and delete files
  - `admin` — full access including member management
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the membership.
</ResponseField>

<ResponseField name="address" type="string">
  The Sui wallet address of the member.
</ResponseField>

<ResponseField name="role" type="string">
  The assigned role (`viewer`, `contributor`, or `admin`).
</ResponseField>

<ResponseField name="driveId" type="string">
  The drive this membership belongs to.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the member was added.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.tusky.io/v2/drives/drive_abc123def456/members \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "0x1a2b3c4d5e6f7890abcdef1234567890abcdef12",
    "role": "contributor"
  }'
```

</CodeGroup>

### Response

```json
{
  "id": "member_stu901vwx234",
  "address": "0x1a2b3c4d5e6f7890abcdef1234567890abcdef12",
  "role": "contributor",
  "driveId": "drive_abc123def456",
  "createdAt": "2025-07-02T10:00:00.000Z"
}
```

<Note>
  Only drive owners and admins can add new members. The invited user's Sui wallet address must be valid.
</Note>
