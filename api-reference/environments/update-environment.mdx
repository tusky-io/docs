---
title: Update Environment
api: 'PUT https://api.tusky.io/v2/environments/{environmentId}'
---

Update the properties of an existing environment.

## Path Parameters

<ParamField path="environmentId" type="string" required>
  The unique identifier of the environment to update.
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  The new name for the environment. Must be between 1 and 128 characters.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the environment.
</ResponseField>

<ResponseField name="name" type="string">
  The updated name of the environment.
</ResponseField>

<ResponseField name="encrypted" type="boolean">
  Whether the environment is encrypted.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the environment was created.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of when the environment was last updated.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X PUT https://api.tusky.io/v2/environments/env_abc123def456 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Renamed Environment"
  }'
```

</CodeGroup>

### Response

```json
{
  "id": "env_abc123def456",
  "name": "Renamed Environment",
  "encrypted": false,
  "createdAt": "2025-07-01T12:00:00.000Z",
  "updatedAt": "2025-07-05T09:15:00.000Z"
}
```

<Note>
  The `encrypted` property cannot be modified after environment creation.
</Note>
