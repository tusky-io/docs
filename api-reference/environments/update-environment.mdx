---
title: Update Environment
api: 'PATCH https://api.tusky.io/v2/environments/{environmentId}'
---

Update an environment's settings. Only included fields are changed.

## Path Parameters

<ParamField path="environmentId" type="string" required>
  The environment ID.
</ParamField>

## Request Body

<ParamField body="name" type="string">
  New environment name.
</ParamField>

<ParamField body="publisherId" type="string">
  Assign a different publisher. Set to `null` to revert to the account default.
</ParamField>

## Response

Returns the updated environment object.

## Example

<CodeGroup>

```bash cURL
curl -X PATCH https://api.tusky.io/v2/environments/env_abc123 \
  -H "Api-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "Staging Assets", "publisherId": "pub_temp"}'
```

</CodeGroup>

### Response

```json
{
  "id": "env_abc123",
  "name": "Staging Assets",
  "encrypted": false,
  "publisherId": "pub_temp",
  "updatedAt": "2026-03-07T15:00:00Z"
}
```

<Note>
  The `encrypted` flag cannot be changed after creation.
</Note>
