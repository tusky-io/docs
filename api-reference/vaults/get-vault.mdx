---
title: Get Vault
api: 'GET https://api.tusky.io/v2/vaults/{vaultId}'
---

Retrieve details of a specific vault by its ID.

## Path Parameters

<ParamField path="vaultId" type="string" required>
  The unique identifier of the vault.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the vault.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the vault.
</ResponseField>

<ResponseField name="encrypted" type="boolean">
  Whether the vault is encrypted.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the vault was created.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of when the vault was last updated.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl https://api.tusky.io/v2/vaults/vault_abc123def456 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</CodeGroup>

### Response

```json
{
  "id": "vault_abc123def456",
  "name": "My First Vault",
  "encrypted": false,
  "createdAt": "2025-07-01T12:00:00.000Z",
  "updatedAt": "2025-07-01T12:00:00.000Z"
}
```
