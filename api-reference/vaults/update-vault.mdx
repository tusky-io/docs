---
title: Update Vault
api: 'PUT https://api.tusky.io/v2/vaults/{vaultId}'
---

Update the properties of an existing vault.

## Path Parameters

<ParamField path="vaultId" type="string" required>
  The unique identifier of the vault to update.
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  The new name for the vault. Must be between 1 and 128 characters.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the vault.
</ResponseField>

<ResponseField name="name" type="string">
  The updated name of the vault.
</ResponseField>

<ResponseField name="encrypted" type="boolean">
  Whether the vault is encrypted.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the vault was created.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of when the vault was last updated.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X PUT https://api.tusky.io/v2/vaults/vault_abc123def456 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Renamed Vault"
  }'
```

</CodeGroup>

### Response

```json
{
  "id": "vault_abc123def456",
  "name": "Renamed Vault",
  "encrypted": false,
  "createdAt": "2025-07-01T12:00:00.000Z",
  "updatedAt": "2025-07-05T09:15:00.000Z"
}
```

<Note>
  The `encrypted` property cannot be modified after vault creation.
</Note>
