## Why Walrus

Walrus is powerful. It makes your data decentralized and verifiable by anyone. Moving to Walrus from any centralized service provider means:

1. **No vendor lock-in.** You can read and write data using locally installed Walrus tooling. You'll need a Sui wallet with tokens (SUI & WAL), plus some RAM and CPU — Walrus encodes your file in memory and distributes it across the network.
2. **Your data is verifiable — and public by default.** Anyone can read it. Do not store private data on Walrus without encrypting it first. If you want decentralized *and* private storage, encrypt before storing and keep the key safe.
3. **You control the lifecycle.** You choose how long a file stays on the network (measured in Walrus epochs) and pay accordingly in WAL tokens. You can extend or delete files at any time.
4. **Simple key-value access.** Every piece of content on Walrus gets a unique identifier (a blobID, or a quiltPatchID for batched uploads). You retrieve content using that identifier.

## Why Tusky

Tusky is an infra provider for [Sui Walrus](https://walrus.xyz). It makes using Walrus easy at any scale.

Tusky gives you managed infrastructure — publishers, private aggregators, encryption, access control, and a familiar file-system-like experience on top of the Walrus network.

## How It Works

Tusky orchestrates **publishers** and **private aggregators** for each user:

- **Publishers** handle encoding and writing your data to the Walrus network. Each user gets a dedicated managed wallet for publishing, funded from your prepaid balance.
- **Aggregators** reconstruct and serve your data from the decentralized network. Tusky runs optimized aggregators with standard URL schemes for seamless access.

You interact with Tusky through the [dashboard](https://dashboard.tusky.io), the [TypeScript SDK](/sdk/getting-started), or the [REST API](/api-reference/overview). Under the hood, Tusky handles blob encoding, storage node coordination, encryption key management, and data retrieval.

<Note>
  Tusky is not a storage network — it is an **infrastructure layer** built on top of Walrus. Your data is stored on the decentralized Walrus network. Tusky provides the tooling, encryption, and access management to make that storage practical for real applications.
</Note>
